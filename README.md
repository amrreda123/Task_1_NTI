# 🚀 NTI Backend Training - Native PHP

Welcome to my Native PHP repository! This project contains all the practical codes, session notes, and assignments I completed during my Backend Development training. It documents my progression from basic PHP syntax to building functional, session-based web applications without a framework.

## 📂 Repository Structure

The repository is organized into two main categories: **Sessions** (lecture practices) and **Tasks** (assignments).

### 📚 Sessions (`Session_One` to `Session_8`)
These folders contain the code written and tested during the lectures, covering fundamental concepts such as:
- PHP Basics (Variables, Data Types, Operators)
- Control Structures (If conditions, Switch)
- Loops (For, While, Foreach)
- Arrays (Indexed, Associative, Multidimensional)
- Superglobals (`$_GET`, `$_POST`, `$_SESSION`)
- File Handling & Data processing

### 💻 Tasks (`task_1` to `task_5`)
These folders contain the practical assignments where I applied the concepts learned in the sessions. 

🌟 **Highlighted Project: Task 5 (Mini E-commerce)**
The latest task (`task_5`) is a fully functional mini e-commerce website built entirely with Native PHP. Features include:
- **Authentication:** Login and Registration system using PHP Sessions.
- **Form Validation:** Server-side validation with Flash Messages to preserve user input.
- **Dynamic Content:** Products are rendered dynamically by decoding a `products.json` file.
- **Clean Architecture:** Separating Logic (PHP Processing) from UI (HTML/Bootstrap Views).

## 🛠️ Technologies Used
- **Backend:** Native PHP
- **Data Storage:** JSON & PHP Sessions
- **Frontend:** HTML5, CSS3, Bootstrap 4, FontAwesome
